==========================================================
    암호화폐 펀딩비 모니터링 텔레그램 봇 사용 설명서
==========================================================

📈 주요 기능
-----------
1. 실시간 펀딩비 모니터링 (Bybit, Binance, Bitget)
2. 현물+선물 교집합 필터링 (더 안전한 투자)
3. 자동 알림 시스템
4. 맞춤형 임계값 설정
5. 거래량 필터링

🤖 봇 명령어
-----------
/start    - 봇 시작 및 자동 알림 활성화
/settings - 알림 설정 변경
/status   - 현재 설정 상태 확인
/now      - 현재 펀딩비 TOP 5 즉시 확인
/stop     - 알림 일시 중지
/resume   - 알림 재개

⚙️ 설정 항목
-----------
1. 펀딩비 임계값: 알림 받을 최소 펀딩비율 (예: 0.1%)
2. 거래량 필터: 최소 거래량 이상 코인만 알림 (예: $1,000,000)
3. 거래소 선택: Bybit, Binance, Bitget 중 선택 가능
4. 현물+선물 필터: 현물과 선물 둘 다 있는 코인만 표시 (권장)

🔔 자동 알림 종류
--------------
1. 정기 리포트 (30분마다)
   - 각 거래소별 양수/음수 펀딩비 TOP 5

2. 임계값 알림 (5분마다)
   - 설정한 임계값 이상의 펀딩비 발생 시 즉시 알림
   - 코인명, 펀딩비율, 현재가, 거래량, 다음 펀딩 시간 포함

💡 현물+선물 필터의 장점
----------------------
- 현물과 선물이 모두 거래되는 코인만 선별
- 유동성이 높고 안전한 코인들로 제한
- 비주류 코인으로 인한 리스크 감소
- 더 안정적인 차익거래 환경 제공

📊 표시 정보
-----------
- 코인명 (예: BTC, ETH)
- 펀딩비율 (양수/음수 %)
- 현재가 (USD)
- 24시간 거래량 (USD)
- 다음 펀딩까지 남은 시간
- 거래소별 차트 링크

🔧 설정 방법
-----------
1. /start 명령어로 봇 시작
2. /settings 으로 설정 메뉴 접근
3. 각 항목별로 원하는 값 설정:
   - 펀딩비 임계값: 0.1 (0.1% 이상)
   - 거래량 필터: 1000000 (100만 달러 이상)
   - 현물+선물 필터: 활성화 권장

⭐ 사용 팁
---------
- 처음 사용 시 임계값을 0.1%로 설정 권장
- 거래량 필터를 통해 유동성 높은 코인만 선별
- 현물+선물 필터는 반드시 활성화 (안전성 증대)
- /now 명령어로 언제든 즉시 확인 가능

🚨 주의사항
----------
- 펀딩비는 변동성이 크므로 신중한 투자 필요
- 높은 펀딩비가 항상 좋은 기회는 아님
- 리스크 관리를 위해 포지션 크기 조절 필수
- 시장 상황에 따라 설정값 조정 권장

📞 문의사항
----------
봇 관련 문의나 버그 발견 시 개발자에게 연락
